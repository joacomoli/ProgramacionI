Ejercicios de Promises - JavaScript Asíncrono

Ejercicio 1: Promise Básica

Objetivo: Crear una Promise que simule una operación asíncrona exitosa.

Conceptos clave:

  • new Promise() - Crear una nueva Promise
  • resolve() - Cumplir la Promise exitosamente
  • reject() - Rechazar la Promise con un error
  • .then() - Ejecutar código cuando la Promise se cumple
  • .catch() - Manejar errores de la Promise

Objetivo: Crear una Promise que puede fallar y manejar el error correctamente.

Conceptos clave:

  • reject() - Rechazar la Promise con un error
  • .catch() - Capturar y manejar errores
  • Parámetros booleanos para simular éxito/fallo
  • Manejo de errores con Error objects

Ejercicio 3: Promise.all

Objetivo: Ejecutar múltiples Promises en paralelo y esperar a que todas se completen.

Conceptos clave:

  • Promise.all() - Ejecutar múltiples Promises en paralelo
  • Todas las Promises deben completarse exitosamente
  • Si una falla, toda la operación falla
  • Resultados en el mismo orden que las Promises

Ejercicio 4: Promise.race

Objetivo: Ejecutar múltiples Promises y obtener el resultado del que termine primero.

Conceptos clave:

  • Promise.race() - Ejecutar múltiples Promises y obtener la primera respuesta
  • Útil para timeouts y conexiones alternativas
  • La primera Promise que se complete gana
  • Las demás Promises se ignoran
