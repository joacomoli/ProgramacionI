Ejercicios de Async/Await - JavaScript Asíncrono

Ejercicio 1: Async/Await Básico

Objetivo: Convertir código con Promises a async/await para mayor legibilidad.

Con Promises (.then/.catch)

Conceptos:

  • .then() para manejar éxito
  • .catch() para manejar errores
  • Código más anidado
  • Difícil de leer con múltiples operaciones

Con Async/Await

Conceptos:

  • async para funciones asíncronas
  • await para esperar Promises
  • try/catch para manejar errores
  • Código más legible y secuencial

Ejercicio 2: Operaciones Secuenciales

Objetivo: Ejecutar múltiples operaciones asíncronas de forma secuencial usando async/await.

Conceptos clave:

  • Operaciones secuenciales - Una después de otra
  • Cada await espera a que termine la operación anterior
  • Útil cuando una operación depende de la anterior
  • Más lento pero más controlado
